# Robotic-Arm-Automatic-Sampling
Scripts and files for running automated nano-DESI experiments using a robotic arm to replace slides automated location of tissue sections.

Included are LabView files to control the motorized XYZ stage, the robotic arm, and the signalling to the Mass Spectrometer, as well as the automatic detection of tissues on sample slides for directing the nano-DESI probes to the desired areas for imaging.

Description of work can be found in "A robotic sample handling platform for fully automated nanospray desorption electrospray ionization mass spectrometry imaging", submitted to J. Am. Soc. Mass Spectrom. in 2026.
